setResource(pType, pName, pPath, pRemote)
pType: type of file to set
pName: name of file to set; this will be referenced in the project; example if pName was set to 'my_atlas' ( atlasName = 'my_atlas' )
pPath: path to the file; if not a remote file this will be the name of the file (or path if any sub directories were added) inside resources/files/ for the game, otherwise it will be a web URL path to the file
pRemote: optional; if set the pPath file will be treated like a web URL
called: sets a game resource to use the resource file located at pPath
