Object:onTick
onTick(pT)
Parameters
pTParameters passed as arguments into the function.
Description
This event function is called every 10ms if an Object has been added to the ticker.
Example Code
Mob onLogin() Event.addTicker(this) // This will add the Mob to the ticker when they login. Mob onTick(pT) World.log(pT) // This will output the ticker count every 10ms.
