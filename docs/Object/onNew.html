Object/onNew
onNew(pArg1, pArg2, ..., pArgn)
Parameters
pArg1, pArg2, ..., pArgnParameters passed as arguments into the function.
Description
This event function is called after an Object has been created.
Example Code
function example() // create a new object with the type 'Object/Test' and pass in the number 45 var d = new Object('Object/Test', 45) // output the object's num value through an alert alert(d.num) Object/Test // this onNew() event function is automatically called when a new 'Object/Test' is created onNew(pNum) // set this object's 'num' value to pNum, which equals 45 from the above code this.num = pNum
