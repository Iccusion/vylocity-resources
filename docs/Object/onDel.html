Object/onDel
onDel(pArg1, pArg2, ..., pArgn)
Parameters
pArg1, pArg2, ..., pArgnParameters passed as arguments into the function.
Description
This event function is called before an Object has been deleted.
Example Code
// get the first MyObject Object found in an imaginary array and delete it var o = World.getObjectsFromArray(myArray, 'MyObject')[0] del Object(o, 45) MyObject // this onDel() event function is automatically called when a 'MyObject' is deleted onDel(pNum) // output the number pNum, which equals 45 from the above code alert(pNum)
