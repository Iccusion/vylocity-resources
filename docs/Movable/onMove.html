Movable:onMove
onMove(pX, pY, pDir)
Parameters
pXThe amount of pixels a Movable is attempting to move on the x-axis.
pYThe amount of pixels a Movable is attempting to move on the y-axis.
pDirThe direction a Movable is attempting to move.
Description
This event function is called when a Movable is attempting to move. It is called before default movement is complete.

If this is defined, return true to continue movement, otherwise movement will be prevented.
Example Code
Movable onMove(pX, pY, pDir) if(pDir == 'west') return false else return true // This will prevent a Movable from moving to the west.
