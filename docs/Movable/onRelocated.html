Movable:onRelocated
onRelocated(pX, pY, pMap, pMove)
Parameters
pXThe Movable's previous xPos (position on the x-axis).
pYThe Movable's previous yPos (position on the y-axis).
pMapThe Movable's previous mapName (placement between maps).
pMoveWhether or not position changed from movement. true if position was changed from movement, false if set directly.
Description
This event function is called when a Movable changes between x or y positions or maps.
Example Code
Movable onRelocated(pX, pY, pMap, pMove) if(pMap != this.mapName) World.log('Movable was moved to map ' + pMap) // If the Movable's map changes, output the new map's name.
