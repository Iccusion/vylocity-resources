Movable/onScreenMoved
onScreenMoved(pClient)
Parameters
pClientA reference to the Client.
Description
This event function is called when a Movable changes position while within a Client's screen view.

This event function is CLIENT-SIDE ONLY.
Example Code
Movable onScreenMoved(pClient) World.log('Movable has moved within the clients screen.') // This will output when the Movable changes positions within a Client's screen.
