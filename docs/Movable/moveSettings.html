moveSettings: object containing settings for moving
{
'stepSize': amount of pixels moved with each step (default of 2)
'stepGlide': determines if the movable glides around edges of solid objects (default true for mobs, false for objects); set to a number to add or remove from the amount of pixels to glide at; normal glide step size is half the movable's width or height (negative values to reduce size)
'stepSlide': determines if the movable changes direction when moving or keeps the direction manually set (default false)
'stepTick': optional; time between each step, default is 60 steps a second for smoothest possible movement (default of 16.6667)
}
