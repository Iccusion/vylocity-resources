Movable:onMoved
onMoved(pX, pY, pDir)
Parameters
pXThe amount of pixels a Movable was moved on the x-axis.
pYThe amount of pixels a Movable was moved on the y-axis.
pDirThe direction a Movable was moved.
Description
This event function is called when a Movable is moved. It is called after default movement is complete.
Example Code
Movable onMoved(pX, pY, pDir) World.log('Movable was moved to the ' + pDir) // This will output which direction the Movable was moved.
