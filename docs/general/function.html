Functions


Description
A function is a block of code designed to perform a particular task.

A function is executed when it is called. It can be reused as many times as needed.

A function is defined with the function keyword, followed by a name, followed by parentheses ().

Objects can hold exclusive functions. To reference an Object's function, the variable referencing the Object has to be put before the function is called.
Inside an Object's function, you can reference the parent Object with the this keyword.

You can obtain an array containing the arguments of the function using the arguments keyword.

When Vylocity reaches a return statement, the function will stop executing.
If the function was called from a statement, Vylocity will resume executing the code after the invoking statement.

Functions often compute a return value. The return value is "returned" back to the "caller".

Function names can contain letters, digits and underscores. Digits cannot be the first character in the name of a function. The same rules apply to variables.
Example Code
function exampleFunction(x, y) return x * y // The function returns the product of arguments x and y. Mob onLogin() World.log(exampleFunction(5, 8)) // This will output 40 to the console when a mob logs in. Diob var a = 5 function someFunc(x, y) World.log(this.a * x * y) // The function returns the product of arguments x, y and this.a Mob onLogin() World.log(this.someFunc(5, 8) // This will output 200.
