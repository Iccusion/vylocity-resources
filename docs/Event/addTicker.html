addTicker(pO, pS)
pO: reference to the object
pS: optional; an object containing settings for this object's ticker
{
delay: optional; default 1; number of ticks before executing
resetAt: optional; default 100; number of ticks to reset the ticker counter at
resetEnd: optional; default false; if set, this object will be removed from the ticker automatically instead of resetting
}
called: add pO to the ticker
