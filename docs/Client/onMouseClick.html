Client/onMouseClick
onMouseClick(pDiob, pX, pY, pButton)
Parameters
pDiobThe Diob that the Client's mouse clicked.
pXThe mouse's position on the x-axis of the Client's screen.
pYThe mouse's position on the y-axis of the Client's screen.
pButtonThe button used; 1 for left, 2 for middle, 3 for right.
Description
This event function is called when the Client's mouse clicks a Diob.
Example Code
Client onMouseClick(pDiob, pX, pY, pButton) // if the client left-clicks a diob if(pButton == 1) // alert the client of information from the event function alert('The client has clicked ' + pDiob.name + ' at the screen position: ' + pX + ', ' + pY)
