Client/onMouseUp
onMouseUp(pDiob, pX, pY, pButton)
Parameters
pDiobThe Diob that the Client's mouse button released on.
pXThe mouse's position on the x-axis of the Client's screen.
pYThe mouse's position on the y-axis of the Client's screen.
pButtonThe button used; 1 for left, 2 for middle, 3 for right.
Description
This event function is called when one of the Client's mouse button have been pressed while over a Diob.
Example Code
Client onMouseUp(pDiob, pX, pY, pButton) // if the client releases the left mouse button over a diob if(pButton == 1) // alert the client of information from the event function alert('The client has moused up over ' + pDiob.name + ' at the screen position: ' + pX + ', ' + pY)
