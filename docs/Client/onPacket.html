Client:onPacket
onPacket(pName, pData)
Parameters
pNameA string containing the name of the packet.
pDataAn object containing information for the packet. ex {'a': 1, 'b': 2}
Description
This event function is called when a Client or server sends a packet using Client.sendPacket.
Example Code
Client // when the client receives a packet onPacket(pName, pData) // alert the client of the packet's name and its 'test' value (if you send one with it) alert('Received packet ' + pName + ': ' + pData.test)
