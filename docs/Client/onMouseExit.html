Client:onMouseExit
onMouseExit(pDiob, pX, pY)
Parameters
pDiobThe Diob that the Client's mouse exited.
pXThe mouse's position on the x-axis of the Client's screen.
pYThe mouse's position on the y-axis of the Client's screen.
Description
This event function is called when the Client's mouse exits a Diob.
Example Code
Client onMouseExit(pDiob, pX, pY) // alert the client of information from the event function alert('The mouse has exited ' + pDiob.name + ' at the screen position: ' + pX + ', ' + pY)
