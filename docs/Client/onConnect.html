Client/onConnect
onConnect(pPar)
Parameters
pParA string containing URL parameters sent when the Client connects.
Loading a game URL such as "http://127.0.0.1:1234/?test" would send "test" into onNew as pPar on both the server-side and client-side.
Description
This event function is called when a Client has successfully connected to the world; if solo world, it is called when the main user loads.
Example Code
Client onConnect(pPar) // send an alert with the parameters included alert(pPar)
