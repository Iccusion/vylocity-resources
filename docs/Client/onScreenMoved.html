Client:onScreenMoved
onScreenMoved(pX, pY, pOldX, pOldY)
Parameters
pXThe screen's new position on the x-axis.
pYThe screen's new position on the y-axis.
pOldXThe screen's old position on the x-axis.
pOldYThe screen's old position on the y-axis.
Description
CLIENT ONLY
This event function is called when the Client's screen moves.
Example Code
Client onScreenMoved(pX, pY, pOldX, pOldY) World.log('The screen moved from ' + pOldX + ', ' + pOldY + ' to ' + pX + ', ' + pY)
