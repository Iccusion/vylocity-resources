Client:onKeyUp
onKeyUp(pKey, pEvent)
Parameters
pKeyA string containing the code for the released key.
pEventCLIENT ONLY The JS KeyboardEvent object.
Description
This event function is called when a key is released.
Example Code
Client onKeyUp(pKey, pEvent) // if the client releases the 'F' key if(pKey == 'f') // alert the client alert('The client released the F key!')
