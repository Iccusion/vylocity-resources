Client:onNew
onNew(pPar)
Parameters
pParA string containing URL parameters sent when the Client connects.
Loading a game URL such as "http://127.0.0.1:1234/?test" would send "test" into onNew as pPar on both the server-side and client-side.
Description
This event function is called when a Client object has been created.

The Client object does not yet have a player Mob at this point, use Client:onConnect if you need a player Mob.
Example Code
Client onNew(pPar) // send an alert with the parameters included alert(pPar)
