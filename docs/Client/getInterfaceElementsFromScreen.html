getInterfaceElementsFromScreen(pX, pY, pX2, pY2, pType, pChild, pPage)
CLIENT ONLY
pX: starting x position
pY: starting y position
pX2: optional; ending x position
pY2: optional; ending y position
pType: optional; string containing type path to return
pChild: optional; boolean that determines if child types are included
pPage: optional; if set, the position will be treated as page position instead of screen position
called: returns an array of diobs on the screen that overlap the specified position box depending on the provided arguments; if pType is not set, all diobs in the position box will be returned; if pType is set but pChild is not, diobs in the position box with the exact type of pType will be returned; if pType is set and pChild is set, all diobs with the type pType or a parent type of pType will be returned; for example, if pType is set to 'Interface' with pChild undefined, all diobs with the exact type of 'Interface' will be returned, but if pChild is set, then all interfaces on the screen will be returned
