Client/onWindowResize
onWindowResize(pWidth, pHeight)
Parameters
pWidthThe width of the Client's window.
pHeightThe height of the Client's window.
Description
This event is called when the Client's window is resized.
Example Code
Client onWindowResize(pWidth, pHeight) // alert the client of the new window size alert('The client has resized the window to: ' + pWidth + ', ' + pHeight)
