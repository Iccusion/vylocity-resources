World/onPost
onPost(pPost, pFrom, pHeaders)
Parameters
pPostA string containing the post data received.
pFromA string containing the address of the sender.
pHeadersAn object containing the sent headers.
Description
This event function is called when a post request is sent to the game server.

World/apiOrigin must be set in order to receive messages; the return value will be written back.

You may return an array to customize the header type. Example: [401, 'return text']
A third value may be included in said array as an object containing headers. Example: {'Content-Type': 'text/html'}
Example Code
World onPost(pPost, pFrom, pHeaders) // Output the post and sender to the console. World.log(pFrom + ' has sent the following data: ' + pPost) // Output the headers object to the console. World.log(pHeaders) // Returning an array customizes the header type. return [401, 'This is the return text', {'Content-Type': 'text/html'}]
