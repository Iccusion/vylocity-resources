onPost(pPost, pFrom, pHeaders)
SERVER ONLY
pPost: string containing the post data received
pFrom: address of the sender
pHeaders: object containing headers sent
called: called when post request is send to the game server; World/apiOrigin must be set in order to get messages; the return value will be written back; you may use an array as the return to customize the header type [401, 'return text'] (a third value may be included which is an object containing headers, for example {'Content-Type': 'text/html'})
