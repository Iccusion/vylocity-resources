World/onMessage
onMessage(pMes,pFrom)
Parameters
pMesA string containing the message received.
pFromA string containing the address of the sender.
Description
This event function is called when another game server or client sends a message to the server.

World/apiOrigin must be set in order to receive messages; the return value will be written back.
Example Code
World onMessage(pMes, pFrom) // log the received information World.log('Received message ' + pMes + ' from ' + pFrom)
