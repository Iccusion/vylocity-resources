World/onDel
onDel(pCallback)
Parameters
pCallback(SERVER ONLY) Reference to a callback function that you will need to call manually if you return true, in order to finish the game closing.
Description
This event function is called when the game is closing and being deleted.

Only synchronous code will run by default, to run asynchronous code (SERVER ONLY) this function will need to return true, then you will need to manually call pCallback when all of your code is finished.
Example Code
function exampleFunc(pFunc) var a = 0 thread(1000) a ++ if(a >= 10) pFunc() // game closes World onDel(pCallback) exampleFunc(pCallback) return true
