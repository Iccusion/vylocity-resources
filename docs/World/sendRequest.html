sendRequest(pLink, pMethod, pHeaders, pData, pFunc)
SERVER ONLY
pLink: string containing the link to send a request to; for example 'https://vylocity.com/'
pMethod: optional; string containing method type of request, ex 'POST', 'GET'
pHeaders: optional; object containing custom headers
pData: optional; data to send with method 'POST'
pFunc: optional; function to call after the request has been sent and a response received, the arguments of the function will be (pData, pError) where pData is the data returned and pError the error (if there is one)
called: sends a request of method pMethod to the url pLink and passes the response into pFunc
