World/onError
onError(pError)
Parameters
pErrorA string containing the thrown error.
Description
This event function is called when debug is enabled and the engine throws an error.

NOTE: Debug is enabled by default, the only way to disable it is by including the code tag #DISABLE DEBUG somewhere in a project.
Example Code
World onError(pError) // output the thrown error to the host console, but it will also already do this by default World.log(pError)
