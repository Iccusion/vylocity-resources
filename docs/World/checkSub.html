checkSub(pName, pFunc)
pName: name of the Vyocity account to check sub of
pFunc: function to call after the sub check; function is called with up to two parameters where the first is true or false depending on if the account is a sub or not and the second is a string containing an error message if there was one
called: checks if Vylocity account pName is a subscriber of the game then calls pFunc; server-side only
