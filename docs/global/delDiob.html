global:delDiob
del Diob(pDiob, pArg1, pArg2, ..., pArgn)
Parameters
pDiobA reference to the Diob to delete.
pArg1, pArg2, ..., pArgnoptionalParameters to pass as arguments into the Diob's onDel() event function.
Returns
undefined
Description
Deletes the Diob referenced by pDiob.

If any of the optional parameters are included, they will be sent into the Diob's onDel() event function as arguments.

Note:This function requires having a reference to a Diob. The best method to get a reference is to store the reference yourself when you create the Diob with global:newDiob(), but many built-in functions can help you get a reference. You can find a list of built-in functions which will assist you in getting a reference at notes:DiobReference.

Note:The Diob will become a blank object and no longer be used by the engine, but any references to the Diob in the code will still exist, so be sure to properly remove the Diob from arrays, stop threads and other events associated with the Diob, and clear any variables that reference the Diob before deleting. The garbage collection will not kick in if these things are not done and the left over object will float around in the memory forever.
Example Code
/* create a new diob and then delete it */ var d = new Diob('Mob') del Diob(d) del Diob(d, 123) // get the first 'Mob/Test' that can be found in the world and delete it var d = World.getDiobs('Mob/Test')[0] del Diob(d, 45) Mob/Test // this onDel() event function is automatically called when a 'Mob/Test' is deleted onDel(pNum) // output the number passed in to the Client that owns this mob, which equals 45 from the above code this.client.alert(pNum)
