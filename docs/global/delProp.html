global:delProp
del Prop(pO, pProp)
Parameters
pOA reference to the object to delete a variable from.
pPropA string containing the name of the variable to delete.
Returns
undefined
Description
Deletes the variable with the name equal to the string held in pProp.
Example Code
/* create a new object and then delete the 'a' property */ var o = {'a': 1, 'b': 2} del Prop(o, 'a') // o = {'b': 2}
