global:newObject
new Object(pType, pArg1, pArg2, ..., pArgn)
Parameters
pTypeA string containing the type path of the Object to be created.
pArg1, pArg2, ..., pArgnoptionalParameters to pass as arguments into the new Object's onNew() event function.
Returns
ObjectThe newly created Object.
Description
Creates a new Object with the specified type of pType and returns it.

If any of the optional parameters are included, they will be sent into the new Object's onNew() event function as arguments.
Example Code
new Object('MyObject') new Object('MyObject/A') new Object('MyObject/B', 123) // create 'MyObject/C' and override the 'myVar' value with 45 new Object('MyObject/C{myVar=45}', 1, 2) function example() // create a new object with the type 'Object/Test' and pass in the number 45 var d = new Object('Object/Test', 45) // output the object's num value through an alert alert(d.num) Object/Test // this onNew() event function is automatically called when a new 'Object/Test' is created onNew(pNum) // set this object's 'num' value to pNum, which equals 45 from the above code this.num = pNum
