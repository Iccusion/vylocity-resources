Server API:
desc: you may communicate with a game server remotely by accessing the game server's api link, which can be found by going to http://yourworldip:port/api
api: current api parameters are 'type'
type: current types are 'message'
message: send a message to the game server; usage ( http://yourworldip:port/api?type=message&text=whatever ); messages can be processed using the World/onMessage event; World/apiOrigin must be set properly to get messages
