CLIENTCODE:
desc: client-side code is code executed on the client-side of a multi-player game; client-side code is specified using the code tag #BEGIN CLIENTCODE and the code can consist of what normal game code can, with the exception that you cannot declare or set World values and you cannot define new object types that do not exist in the normal code. Client code can communicate with the server code, and vice versa, by using the client.sendPacket function.
