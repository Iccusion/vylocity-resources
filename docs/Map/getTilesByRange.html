getTilesByRange(pD, pVal1, pVal2, pType, pChild, pOmit)
pD: diob or range object to get range from; range object example {'map': 'name', 'x': 1, 'y': 1}; using a range object will use the tile at the specified location for the reference diob
pVal1: either x distance or x and y distance
pVal2: if set, y distance
pType: type or parent type of tiles to get
pChild: true to include all child tiles; false to include only tiles with the exact type of pType
pOmit: optional; if set, pD will not be included in the results
called: returns an array containing all the tiles on the map within x and y pixels around the specified diob
