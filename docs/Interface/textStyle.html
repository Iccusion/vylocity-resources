textStyle: a JSON object containing styles for the text (ex: {"padding": 0, "fill": "#000", "leading": 0} )
{
fontFamily: default font to be used for text displays
fontSize: default size of font in pixels
textBaseline: where vertically to start drawing the characters on the line; top/alphabetic/hanging/middle/ideographic/bottom
fill: default color of text for this element
stroke: style of text outline for this element; currently supports colors
strokeThickness: width of text outline
padding: padding between the edge of the element and the text
hPadding: padding on left and right sides; overrides padding
vPadding: padding on top and bottom; overrides padding
leading: default spacing between each line of text
bitmap: name of the icon atlas to use as a bitmap font
bitMask: true to turn bitmap text into appearMask icons, the text will apply its parent's appearance properties
charSpace: bitmaps only; default spacing between each character
align: default 'left'; possible values 'left', 'right', 'center', 'start', 'end'; bitmaps only support 'left' or 'right'
}
