Interface:onRelocated
onRelocated(pX, pY)
Parameters
pXThe Interface object's previous xPos (position on the x-axis) on the Client's screen.
pYThe Inetrface object's previous yPos (position on the y-axis) on the Client's screen.
Description
This event function is called when an Interface object changes x or y positions.
Example Code
// define an interface object Interface example onRelocated(pX, pY) // send an alert with information on the interface object's old position alert('this interface object has moved from ' + pX + ', ' + pY)
