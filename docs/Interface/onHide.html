Interface:onHide
onHide(pClient)
Parameters
pClientThe Client the Interface object is being hidden from.
Description
This event function is called after an Interface object has been hidden from a Client's screen.
Example Code
// define an interface object Interface inputBox interfaceType = 'CommandInput' // send an alert to the client onHide() alert('this interface object has been hidden from the client') // hide the 'inputbox' interface object in the 'chat' interface from the client when they press the 'f' key Client onKeyDown('f') this.hideInterface('chat', 'inputbox')
