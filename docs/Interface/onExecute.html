Interface:onExecute
onExecute(pClient)
Parameters
pClientThe Client executing the interface.
Description
This event function is called when the general purpose of an Interface object is executed.

Interface objects with the interfaceType 'CommandInput' are executed when the enter key is pressed while it is has a Client's focus.

Any Interface object can be executed via the Interface.execute() function.
Example Code
// define an interface object Interface inputBox interfaceType = 'CommandInput' // send the text of the interface object to all client's 'outputbox' object within their 'chat' interface. onExecute() World.outputText(this.text, 'chat', 'outputbox')
