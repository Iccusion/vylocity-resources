cacheImage(pAtlas, pIcon, pFunc) CLIENT ONLY
pAtlas: name of the icon atlas
pIcon: optional; name of the icon
pFunc: optional; function called after the images have been loaded and cached
called: loads all the images for each specified icon; icon images are loaded when an icon or state is shown to a client, if you plan on altering an icon that the client hasn't seen yet, you'll want to use this function to cache it first
