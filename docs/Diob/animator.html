animator: an object {'speed': 2, 'reverse': true, 'isPaused': false, 'keepFrame': true} containing information about how this diob's icon is animated
{
speed: speed multiplier for frames
reverse: play animation backwards
isPaused: pause animation
keepFrame: do not reset frame when changing icon/state
sync: if set, will share animation with all other diobs with the same sync id; ex {'sync': 'water'}
}
