Diob.playAnimation
playAnimation(pAtlas, pIcon, pState, pFrame, pEnd, pSet, pFunc)
Parameters
pAtlasA string containing the file path of the icon for the animation.
pIconoptionalA string containing the name of the icon for the animation.
pStateoptionalA string containing the name of the state of the icon for the animation.
pFrameoptionalA number defining which frame to start the animation on. Default: 1
pEndoptionalA number defining which frame to end the animation after. Default: frames.length
pSetoptionalAn object with any animator settings, also including 'loop', 'delay', and 'rFrame'.
The 'loop' setting defines the number of times to loop the animation before stopping.
A 'loop' value of -1 will make it loop until the animation is manually halted.
The 'rFrame' setting defines which animation frame to return to when the animation ends.
Example: ({'loop': -1, 'speed': 2})
pFuncCLIENT-ONLY optionalA function to call after the animation has ended.
Returns
undefined
Description
Sets the Diob's icon and plays the animation for that icon, returning to the Diob's previous icon (or new icon if changed during animation) when the animation has ended.

Calling this function with no parameters will stop the Diob's current animation.
Example Code
Mob onLogin() this.playAnimation('icons', 'player', 'test', 1, 5, {'loop': 10, 'speed': 2}, this.endAnim) function endAnim() World.log('Animation successfully ended!')
