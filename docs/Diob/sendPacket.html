sendPacket(pName, pData, pClient, pForce)
pName: name of packet to send
pData: data to be sent; ex {'a': 1, 'b': 2} or 'test'
pClient: SERVER ONLY; optional; specific client or array of clients to send this packet to; if no client is specified the packet will be sent to all clients currently synced with this diob
pForce; SERVER ONLY; optional; if set, this packet will be sent immediately instead of added to the packet delay system and sent there; this can only be used if pClient is set
called: sends a packet to a client or server specifically for this diob, depending on where the code is executed; client code executing this will send the packet to the server; server code executing this will send the packet to the client; this will invoke Diob/onPacket on client-side if the sendPacket is server-side and on the server-side if it is on the client-side
