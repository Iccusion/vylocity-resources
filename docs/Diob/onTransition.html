onTransition(pVar, pChange, pStart)
pVar: a string containing the name of the variable involved in the transition
pChange: the amount changed
pStart: if this is set then this is the event called when a transition starts; this will be a reference to the transition object used
called: when this diob has a transition change; if pVar, pChange, and pStart are all unset then this is the event called when all transitions have ended
