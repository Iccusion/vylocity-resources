onMapChange(pMap, pBefore)
pMap: name of the old map or name of the new map if pBefore is set
pBefore: if set then the map is currently loading and this is the pre-load event
called: when this diob changes maps; this event will be called twice if the map has not already been loaded, once before loading and once after the loading has completed (the first event will have pBefore set)
