setPattern(pX, pY, pWidth, pHeight)
CLIENT ONLY
pX: default 0; position where the pattern starts on the x-axis
pY: default 0; position where the pattern starts on the y-axis
pWidth: optional; width of the pattern
pHeight: optional; height of the pattern
called: turns this diob's sprite into a repeating pattern; call with no arguments to return to a normal sprite
