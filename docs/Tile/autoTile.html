Tile/autoTile
Tile
  autoTile = 16
Default Value
undefined
Variable Type
number
Description
Autotiling is a system which, if enabled, automatically checks the surrounding tiles of a placement and changes each tile's iconState to be appropriate for its connection to its surrounding tiles. Autotiling allows you to more quickly draw your levels.

autoTile can be assigned to the values of 48, 16, or 1, depending on the amount of states you want to use.
autoTileBlend can be a type to autotile with that is a different type than the Tile. This is useful for a base type.
Assigning a Tile's autoTile to 1 will not autotile itself, but will force surrounding Tiles to autotile if they use autoTileBlend with this Tile's type.

Check out this demo to see the required organization for autotiles to be used.
Example Code
Tile atlasName = 'icon1' iconName = 'grass' Dirt iconName = 'dirt' // this tile wont autotile itself, but can force other tiles around it to autotile if they use autoTileBlend with this type autoTile = 1 DirtGrass48 iconName = 'dirt_grass_48' // this tile has 48 iconStates autoTile = 48 autoTileBlend = 'Tile/Dirt' DirtGrass16 iconName = 'dirt_grass_16' // this tile has 16 iconStates autoTile = 16
